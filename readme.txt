== Description ==

Do you want your visitor to have access into your latest facebook post instantly, add a fixed sidebar all over your website 
and show them your social presence.
With Floating Facebook Page, your facebook page will have a better impact with your user, by allways showing them what you do, in a nice way.

*Easy to setup your short news*
Floating Facebook Page is a plugin to help you integrate your facebook page into your website.

*Easy to setup*
Floating Facebook Page has a nice and clean settings page, to help you integrate your facebook page.

== Installation ==

Floating Facebook Page can be installed like any other wordpress plugin.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/floating-facebook-page` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Bolt News screen to configure the plugin
